# Grocery Sales & Inventory Management System

## Overview

The Grocery Sales & Inventory Management System is a desktop-based application developed using Python, Tkinter, and MySQL. It helps grocery stores manage products, inventory, employees, suppliers, and sales transactions efficiently.

The system automates inventory tracking, billing, and record management to reduce manual errors and improve daily store operations.

## Features

### Admin Features

* Secure login authentication
* Dashboard with business statistics
* Employee management
* Product management
* Supplier management
* Inventory monitoring
* Stock-in and stock-out recording
* Sales reporting
* Archive and restore records

### Cashier Features

* Secure login authentication
* Product search
* Shopping cart management
* Automatic bill generation
* Receipt printing and saving
* Real-time stock updates after sales

## How the System Works

1. Users log in using their credentials.
2. The system verifies the user's role (Admin or Cashier).
3. Admins manage products, suppliers, employees, and inventory.
4. Cashiers process customer purchases.
5. The system automatically:

   * Calculates totals
   * Generates receipts
   * Updates inventory levels
   * Saves transaction records
